# 助手笔记

## 包管理器
- 使用 `yarn` 而不是 `npm`
- 安装包时通常需要 `sudo` 权限
- 存在 `package-lock.json` 时会有警告，但可以忽略

## 网络访问
- 访问外部资源时需要设置代理
- 代理设置命令：
  ```bash
  export http_proxy=http://127.0.0.1:7890 && export https_proxy=$http_proxy
  ```
- 所有需要网络访问的命令都需要先设置代理

## 权限相关
- 某些操作可能需要 `sudo` 权限
- 特别是涉及 `node_modules` 的操作
- 如果遇到权限问题，可能需要：
  1. 使用 `sudo` 运行命令，并提示我输入密码
  2. 修复目录权限
  3. 清理并重新安装 `node_modules`

## 开发环境
- 操作系统：macOS (darwin 24.4.0)
- Shell：/bin/zsh
- 工作目录：/Users/yanfang/dev/markdown-reader

## 项目特定要求
- 使用 TypeScript
- 使用 React + Vite
- 使用 Ant Design 组件库
- 所有数据存储在 localStorage 中
- 支持 Markdown 渲染和 LaTeX 公式
- 需要支持文件上传和本地存储
- 需要支持自定义样式设置

## 注意事项
- 在修改代码前先检查现有文件内容
- 注意处理 TypeScript 类型问题
- 保持代码风格一致性
- 优先使用中文注释和提示信息
- 所有组件和功能都需要考虑移动端适配
